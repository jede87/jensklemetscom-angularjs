chocolatey

first of all i really like using chocolatey. it is a good packet manager for windows like apt-get in linux. i will do all the program installation steps with chocolatey, so please install it if you want a faster ride.

Chocolatey getting started docs

here is the command that you can paste in your command line tool to install chocolatey:
@"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"

cmder

as command line tool i use cmder. it is really smooth and nice to use. it will also remember all the commands you execute.

choco install cmder -y

once installed start cmder with administrator priviliges by right clicking cmder and choose "Run as Administrator". now you can start installing stuff
